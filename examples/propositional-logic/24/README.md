# 24 - Elimination of ∨

Each of the sentences in `GentzenSentences1` involve the use of both `∧` and `∨`.
In each even numbered position write a sentence equivalent to the one above it,
but that uses only `∧` and `¬`.
Check that the truth values of the pairs of sentences agree in a number of worlds.

## Historical: who was Gentzen?

German logician (1909 - 1945)

Student of Paul Bernays and assistant to David Hilbert.

Big contributions to proof theory and foundations of mathematics.
