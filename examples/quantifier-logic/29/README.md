# 29 - Describing a world

- You will describe some features of `PeanoWorld` in `sentencesQ29`.

1. Notice that every circle is small and lime. Use your first sentence to say this.
2. State the fact that there is a medium sized square.
3. Next, assert that there are at least two squares.
4. Express the fact that there is a triangle between two circles.
5. Notice that it is not the case that every square is below a circle. Say this.

- Save your list of sentences as `sentencesQ29`.
- Be sure to check each of your sentences
  to see that it is indeed a sentence and is true.
- Now let's change the world so that none of the above facts hold.
- We can do this by first changing the medium square into a circle,
- and then moving the leftmost circle to the bottom row.
- If your answers to 1-5 are correct, all of the sentences should now be false.
