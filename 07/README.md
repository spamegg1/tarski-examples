# 07 - Translating

- Enter translations of the following English sentences into FOL in `sentences07`.
- You will only need to use the connectives ∧, ∨, and ¬.
- We will check them in the next example.

1. Either a is small or both c and d are large.
2. d and e are both above b.
3. d and e are both above b and larger than it.
4. Both d and c are squares, however neither of them is small.
5. Neither e nor a is to the right of c and to the left of b.
6. Either e is not large or it is above a.
7. c is neither between a and b, nor below either of them.
8. Either both a and e are triangles or both a and f are.
9. Neither d nor c is below either c or b.
10. c is either between d and f or smaller than both of them.
11. It is not the case that b is in the same row as c.
12. b is in the same column as e,
    which is in the same row as d, which in turn is in the same column as a.
