# 22 - Consequence

These two English sentences are consequences of the ten sentences in `BuridanSentences`.

1. There are no squares.
2. There is exactly one big triangle.

Because of this, they must be true in any world in which `BuridanSentences` are all true.
So of course they must be true in `worldQ21`, no matter how you built it.

- Translate the two sentences, adding them to `twoSentencesQ22`.
- `sentencesQ22` contains `BuridanSentences` with your two new sentences added.
- We also copied the solution from Example Q21 into `worldQ22`.
  If you want to use your own solution world, you can copy that one instead.
- Verify that `sentencesQ22` are all true in `worldQ22`.
- Modify the world by adding a square.
- Try placing it at various locations and giving it various sizes
  to see what happens to the truth values of the sentences.
- One or more of the original ten sentences will always be false,
  though different ones at different times.
- Find a world in which only one of the original ten sentences is false.
- Next, get rid of the square and add a second big triangle.
- Again, move it around and see what happens to the truth values of the sentences.
- Find a world in which only one of the original ten sentences is false.
