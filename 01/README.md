# 01 - Wittgenstein's World and Sentences

1. Run the world with `@main def run01`, but do not click "Eval" yet.
2. Observe the blocks and the atomic sentences.
3. Mentally assess the truth values of the sentences.
4. Click "Eval" to see the correct interpretation.
5. If your interpretation is wrong on any of them,
   try to figure out how your interpretation is different than the correct one.
