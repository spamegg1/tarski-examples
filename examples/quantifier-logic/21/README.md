# 21 - Building a world

- Open `BuridanSentences` in `worldQ21`.
- Build a world in which all ten sentences are true.
- Once all ten sentences are true, take a screenshot of your world.
- Close the world and write down the blocks from the screenshot in `worldQ21`.
- Save your world. Run and evaluate again to make sure it's correct.
