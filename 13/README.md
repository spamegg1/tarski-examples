# 13 - Building a world

- Build a world in which all of the English sentences listed in 12 are true.
- Run `run13` to make sure that all your FOL translations are also true.
- If one of your translations is false, see whether the original English sentence is true.
- If it is, then there is something wrong with your translation.
- Play the game (with `play13`) to try to figure out what the problem is.
