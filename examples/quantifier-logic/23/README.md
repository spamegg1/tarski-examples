# 23 - Independence

- Show that the following sentence is independent of those in `BuridanSentences`,
- that is, neither it nor its negation is a consequence of those sentences:

    `∃x ∃y (¬Eq(x, y) ∧ Tri(x) ∧ Tri(y) ∧ Mid(x) ∧ Mid(y))`

- You will do this by building two worlds:
  - one in which this sentence is false (call this `worldQ23a`)
  - and one in which it is true (`worldQ23b`)
  - but both of which make all of `BuridanSentences` true.
- We copied the world from Example Q21 as a starting point.
  - In this world, all `BuridanSentences` are already true, so you don't have to re-do it.
  - You can use it for both parts, then change them in different ways if needed.
- Take screenshots if needed, then write your blocks and save your worlds.
